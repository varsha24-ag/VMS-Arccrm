# ArcCRM: Enterprise Visitor Management

![ArcCRM Banner](https://img.shields.io/badge/Status-Active-emerald?style=for-the-badge)
![Next.js](https://img.shields.io/badge/Frontend-Next.js%2014-black?style=for-the-badge&logo=next.js)
![FastAPI](https://img.shields.io/badge/Backend-FastAPI-009688?style=for-the-badge&logo=fastapi)

**ArcCRM** is a state-of-the-art Visitor Management System designed to bridge the gap between facility security and professional hospitality. Built with MNC-standard aesthetics and enterprise-grade performance, it provides a seamless interface for visitors, employees, and administrative teams.

---

## 💎 Core Value Propositions

### 🛡️ Intelligent Security
Strict identification protocols including international phone validation (10-15 digits) and secure host-approval workflows to ensure only authorized personnel enter the premises.

### 🎨 Premium Experience
A stunning, light-themed user interface powered by **Sora** typography and modern CSS standards, offering a "SaaS-native" feel that wows users at first glance.

### 📊 Administrative Intelligence
Real-time analytics for traffic monitoring, department-wise visitor distribution, and peak-hour facility usage tracking.

### 🌐 Scalable Architecture
Clean separation of concerns with a high-performance **FastAPI** backend and a reactive **Next.js** frontend, ready for third-party authentication and cloud deployment.

---

## 🛠️ The Tech Stack

### Frontend
- **Framework**: `Next.js 14` (App Router)
- **Styling**: `Tailwind CSS` (Custom Design System)
- **Language**: `TypeScript` (Strict Type Safety)
- **Typography**: `Sora` (Google Fonts)

### Backend
- **Engine**: `FastAPI` (Python 3.9+)
- **ORM**: `SQLAlchemy` with `SQLite`
- **Security**: `JWT` + `PassLib` for robust authentication

---

## 🚀 Getting Started

### 1. Prerequisites
- Node.js (v18+)
- Python 3.9+

### 2. Launch Backend
```bash
cd backend
pip install -r requirements.txt
python3 run.py  # Service running on port 8005
```

### 3. Launch Frontend
```bash
cd frontend
npm install
npm run dev     # Access via http://localhost:3000
```
